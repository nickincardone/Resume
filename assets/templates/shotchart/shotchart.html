ShotChart App

Proof of concept of shotchart application, using data from Hawks/Pacers series of 2014 playoffs.

How To Use

Pick An Offensive Player

Picking a player shows all the shots he took in the series. Empty circles indicate a miss while a blue cirlcle is a made two and a red circle is a made three. Select a team to see all the shots from that team during the series.

Pick A Defensive Player

Picking a defesive player shows all the shots the offensive player took while the selected defender was on the floor. Select "All Defenders" to see all the shots the offensive player took.

How It Was Made

I created this by scraping shot chart and play-by-play data with python and converting it to JSON format. I then created this web page to run through the JSON to create the shot charts dynamically.
